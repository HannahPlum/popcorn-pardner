Popcorn Pardner will be a movie reviewing app where users may search for a movie, write a review for it that will be posted to their profile page, assign the movie a rating, and add comments. Users will be able to add friends and see what they're watching/posting.

Primary features will include:

1. Search - The user may search for a movie (TMDB API)
2. Watchlist - The user may add movies they haven't seen yet to a watchlist
3. Movie review - the user may "log" a movie, leaving a review with their thoughts to be posted to their account
4. Movie rating - The user may assign the movie a rating 1-5 in cowboy hats (Yeehaw)
5. Friends - The user should have the ability to follow other users. Once following, their reviews appear in a designated friend's review section
6. Comments - The user may comment on their movie review or on the reviews of friends
7. Currently watching - The user may select "currently watching" on a movie and this will show a live indicator on their profile.

- The front end stack will consist of HTML for the structure, CSS for styling, and JS for any interactivity.

- Movie data will come from The Movie DataBase (TMDB) API

- Firebase Authentication will be used for signups/login/logout, and Firestone will be used to store data
